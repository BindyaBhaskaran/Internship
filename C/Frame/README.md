## Question
Write a function that takes a list of strings and prints them one per line in a  rectangular frame.

            Eg: List: [“Hello”,”World”,”in”,”a”,”Frame”]

            Output:  *********
                     * Hello *
                     * World *
	                   * in    *
	                   * a     *
	                   * frame *
                     *********
                    
## How to compile
    gcc Frame.c -o frame
    
## How to run
    ./frame
